# Roadmap

This document outlines planned features and milestones for MikroClaw.

- Lua skill runtime
- Skill marketplace (cloud-based)
- Discord/Slack channels (full bot support beyond webhooks)
- Package managers (Homebrew, APT)

## [1.0.0] - Planned

- Production hardened
- Full test coverage
- Security audit passed
- Enterprise features
